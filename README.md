# NOVA 品牌官网

一个简约现代的品牌官网单页 HTML，适用于创意工作室 / 品牌设计公司展示。

## 怎么跑

**零依赖，直接打开浏览器即可：**

```bash
# 用浏览器打开（任选一种）
open index.html          # macOS
xdg-open index.html      # Linux
start index.html         # Windows
```

或者直接拖到浏览器窗口里。

## 预期效果

| 模块 | 内容 |
|------|------|
| 顶部导航 | 固定导航栏 + 移动端汉堡菜单 |
| Hero | 品牌标语 + 双按钮 CTA |
| 关于 | 品牌简介 + 数据统计卡片 |
| 服务 | 三张服务卡片（品牌策略 / 视觉识别 / 界面设计）|
| 联系 | 联系入口 + 邮件按钮 |
| 页脚 | 版权信息 + 社交链接 |

- ✅ 响应式设计（桌面 / 平板 / 手机均适配）
- ✅ 滚动渐入动画（IntersectionObserver）
- ✅ 平滑滚动
- ✅ 无外部依赖，单文件即开即用
- ✅ 移动端汉堡菜单交互

## 技术栈

纯 HTML5 + CSS3 + JavaScript（ES6），零第三方依赖。
